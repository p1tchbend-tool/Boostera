# Boostera
「Boostera」は「TeraTerm」をより使いやすくするための外部アプリです。  
使用にはTeraTermが必要となりますので、[公式リポジトリ](https://github.com/TeraTermProject/osdn-download/releases)より別途インストールしてください。  
またSFTP機能を利用する場合は「WinSCP」が必要となりますので、[公式サイト](https://winscp.net/eng/download.php)より別途インストールしてください。  
  
Boosteraは以下の機能をTeraTermに提供します。  
* TTLマクロファイルの高速な検索と実行
* SSH接続の履歴管理
* ウィンドウレイアウト管理
* ポートフォワーディングを利用した踏み台経由の接続
* SSH、SFTP、リモートデスクトップの相互運用

## インストール
1. インストーラーはありません。  
[アプリ本体](https://github.com/p1tchbend-tool/Boostera/tree/master/App)を適当な場所に配置して実行します。

## セットアップ
1. タスクトレイのアイコンをクリックするか「Alt + T」を押してアプリを表示します。
1. 歯車のアイコンをクリックして設定画面を開きます。
1. 「ttermpro.exe」「ttpmacro.exe」の場所を指定します。  
その他項目も必要に応じて設定します。

## アンインストール
1. セットアップ時に「スタートアップに登録する」のチェックを入れた場合はチェックを外します。
1. アプリを終了します。
1. アプリをフォルダごと削除します。
