# Boostera
「Boostera」は、そのままでも便利な「TeraTerm」をより使いやすくするためのツールです。  
以下の機能を提供します。  
  
* TTLマクロファイルの高速な検索と実行
* SSH接続の履歴管理
* 接続履歴をTTLマクロとしてエクスポート
* SSH、SFTP、リモートデスクトップの相互運用
* ポートフォワーディングを利用した踏み台経由の接続
* ウィンドウレイアウト管理
  
使用には「TeraTerm」が必要となりますので、[公式リポジトリ](https://github.com/TeraTermProject/osdn-download/releases) より別途インストールしてください。  
またSFTP機能を利用する場合は「WinSCP」が必要となりますので、[公式サイト](https://winscp.net/eng/download.php) より別途インストールしてください。  

## インストール
1. インストーラーはありません。  
[アプリ本体](https://github.com/p1tchbend-tool/Boostera/releases) を適当な場所に配置して実行します。

## セットアップ
1. タスクトレイのアイコンをクリックするか「Alt + T」を押してアプリを表示します。
1. 歯車のアイコンをクリックして設定画面を開きます。
1. 「ttermpro.exe」「ttpmacro.exe」の場所を指定します。  
その他項目も必要に応じて設定します。
  
以上で使用準備は完了です。  
各機能の説明については、画面にマウスカーソルを当てると表示される、ポップアップを参照してください。

## アンインストール
1. セットアップ時に「スタートアップに登録する」のチェックを入れた場合はチェックを外します。
1. アプリを終了します。
1. アプリをフォルダごと削除します。
  
また以下のフォルダを必要に応じて削除します。  
※本アプリの設定等が保存されているため、再インストールする場合は削除しないでください。  
```
%LOCALAPPDATA%\Boostera
```
